MarkdownRegexOptions

Defines .NET-style synonyms for NSRegularExpressionOptions values

  • Multiline
  • IgnorePatternWhitespace
  • Singleline
  • IgnoreCase
  • None

Note: NSRegularExpressionOptions does not provide equivalents to these .NET RegexOptions values used in the original C# source:

  • Compile
  • ExplicitCapture
  • Multiline

    Allow ^ and $ to match the start and end of lines.
